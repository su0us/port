Main

ALZI의 메인 페이지입니다.
ALZI의 메인 페이지는 정직하고 명확한 ALZI의 브랜드 정체성을 강조하여
블루색을 메인 컬러로 사용하여 차분한 웹 페이지를 구상하였고, 다소 지루해보이지
않도록 GIF파일을 첨부하고, 활기가 넘치는 옐로우색을 서브 컬러로 사용하였습니다.

sub

ALZI의 회사소개 페이지입니다.
회사소개 AI중에서도 대표적으로 Brand Identity 페이지를 구상하였습니다.
메인 페이지와 서로 잘 어울려 조화롭도록 제작하였습니다.

sub

ALZI의 APP소개 페이지입니다.
이용자들에게 회사의 대표적인 서비스인 위치기반 알람 서비스앱의
사용법을 보기좋게 정리해놓은 소개 페이지입니다.
기존에 디자인 되어 있었던 ALZI앱의 이미지를 사용하여 차분하게 퍼블리싱하였습니다.

sub

ALZI의 고객센터 페이지입니다.
고객센터 페이지중에서도 대표적으로 FAQ페이지를 구상하였습니다.
ALZI의 BI에 어울리는 차분한 색감과 아이콘을 사용하여 사용자들이
보다 간편하고 편리하게 문의 할 수 있는 페이지를 퍼블리싱하였습니다.

sub

ALZI의 공지사항 페이지입니다.
공지사항 페이지중에서 소식&이벤트 페이지를 구상하였습니다.
ALZI의 BI에 맞는 정직한 이미지를 사용하였고, 마우스 Hover값을
사용하여 다소 따분하지 않도록 퍼블리싱하였습니다.
